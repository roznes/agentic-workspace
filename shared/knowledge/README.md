# Knowledge Base

Persistent memories for AI agents, organized by topic.

## How to Use

**User:** "Remember: [topic] - [details]"
**AI:** Creates or updates `shared/knowledge/[topic]/README.md`

## Structure

```
knowledge/
├── README.md              # This index
├── [topic-name]/
│   ├── README.md          # Main knowledge document
│   └── [related-files]    # Scripts, configs, etc.
└── ...
```
